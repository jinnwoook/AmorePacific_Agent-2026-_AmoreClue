# 제품 이미지 파일 가이드

## 필요한 이미지 파일 (3개)

다음 파일명으로 이미지를 저장하세요:

### 1. 첫 번째 제품 이미지
**파일명:** `cerave-레티놀-앰플-1.jpg` (또는 `.png`)
- **브랜드:** CeraVe
- **제품명:** 레티놀 앰플
- **설명:** 레티놀과 앰플이 조합된 제품 이미지

### 2. 두 번째 제품 이미지
**파일명:** `the-ordinary-앰플-모공-케어-2.jpg` (또는 `.png`)
- **브랜드:** The Ordinary
- **제품명:** 앰플 모공 케어
- **설명:** 앰플과 모공 케어가 조합된 제품 이미지

### 3. 세 번째 제품 이미지
**파일명:** `la-roche-posay-레티놀-모공-케어-3.jpg` (또는 `.png`)
- **브랜드:** La Roche-Posay
- **제품명:** 레티놀 모공 케어
- **설명:** 레티놀과 모공 케어가 조합된 제품 이미지

## 파일명 규칙

1. 브랜드명과 제품명을 하이픈(-)으로 연결
2. 공백은 하이픈으로 변환
3. 소문자로 작성
4. 마지막에 번호 추가 (1, 2, 3)

## 이미지 권장 사양

- **형식:** JPG, PNG, WebP
- **크기:** 200x200px 이상 (정사각형 권장)
- **비율:** 1:1 (정사각형)

## 예시 파일명

```
cerave-레티놀-앰플-1.jpg
the-ordinary-앰플-모공-케어-2.jpg
la-roche-posay-레티놀-모공-케어-3.jpg
```

이미지가 없으면 자동으로 플레이스홀더가 표시됩니다.
